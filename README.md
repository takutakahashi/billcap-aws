# billcap-aws
Billing capture and transfer for AWS Cost Explorer

## Prerequirements

1. AWS Credentials
2. Data store
  1. BigQuery
2. Execution Environment
  3. k8s

## Usage

### 1. Set AWS Token

ENV `AWS_ACCESS_KEY_ID` and `AWS_SECRET_ACCESS_KEY`

### 2. Set Google Cloud Credentials

ENV `GOOGLE_CLOUD_SERVICE_ACCOUNT_KEY_PATH`
